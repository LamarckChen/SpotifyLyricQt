## 歌词源

* QQ音乐
* 网易云音乐
## 安装
在Release页面下载安装即可，windows安装之后桌面会显示SpotifyLyric图标，在电脑重启之后，会在菜单栏->Spotify Lyric找到本软件。
## Ubuntu下卸载本软件
`sudo dpkg -r spotifylyricQt`
